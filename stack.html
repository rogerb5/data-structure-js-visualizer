A Stack

Is a data structure that adds and removes based on the Last In First [LIFO] principle.

Additionally, elements are stored in order of insertion. We do not think of stacks having indexes instead they have a bottom and top.

The client can only add, delete, and examine the last element at the top of stack

The indexes are shown on the visualizer, but conceptually we think of a stack having a bottom and top.

Basic Stack Operations

Push: add element to the top

Pop: remove the top element

Peek: examine the top element

Applications of Stacks in Computer Science

Programming languages and compilers: methods are places on a stack call means push and return is pop

Matching up related pair of things: check if a string is a palindrome, convert "infix" expression to pre or postifx

Algorithms: search a maze using "backtracking"
